# React 学习路径
* 1.看官方文档

    + [react官方文档-英文原版](https://reactjs.org/docs/hello-world.html)
    + [react官方文档-国内翻译 ](https://doc.react-china.org/docs/hello-world.html)

* 2.写一些小Demo
    + 推荐阮老师的react入门教程  [React 入门实例教程](http://www.ruanyifeng.com/blog/2015/03/react.html)

    去github上把他的例子拷贝下来，一个一个的改一下。 
    会改就证明看懂了这些例子,看不懂的也不急，阮老师对每个例子在他的博文中都有讲解，对着解疑排惑就行。

* 3.学习redux
    + 官方文档：[redux文档](https://redux.js.org/)
    + 阮老师入门教程：[Redux 入门教程（一）：基本用法](http://www.ruanyifeng.com/blog/2016/09/redux_tutorial_part_one_basic_usages.html)

    知道redux的基本概念和用法后，就去官方文档上拷贝的官方提供的几个[栗子](https://github.com/reactjs/redux/tree/master/examples)

    从计数器栗子入手理解redux分层的数据流动。    
    相关学习资料：

    + [1.从零学习Redux](https://yubolun.com/react-learn-3-1/)

    + [2.simplest-redux-example](https://github.com/jackielii/simplest-redux-example/blob/master/index.js)

    当然也可以看下我的博客记录^_^

    + [[一步一步学react系列] 01—学习路径规划](http://blog.csdn.net/young_emily/article/details/78939191)

    + [[一步一步学react系列] 02—计数器V0.0](http://blog.csdn.net/young_emily/article/details/78939409)

    + [ [一步一步学react系列] 03—计数器V1.0](http://blog.csdn.net/young_emily/article/details/78939642)

    + [ [一步一步学react系列] 04—计算器Demo](http://blog.csdn.net/young_emily/article/details/79021954)
